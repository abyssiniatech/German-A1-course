🅴 PART 5: TRUE / FALSE – 81–90

“Kind” uses das → ✅ True

Plural nouns use die → ✅ True

“Familie” uses der → ❌ False

“Mutter” uses die → ✅ True

“Haus” uses das → ✅ True

Countries always use articles → ❌ False

“Bruder” uses der → ✅ True

“Baby” uses die → ❌ False

Objects are often das → ✅ True

Singular nouns can have no article → ❌ False
